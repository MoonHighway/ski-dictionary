Skier Dictionary
================
A simple Express site that can be used to teach Node js.

Topics
------

* Express
* Middleware
* Routing
* Browserify
* Testing
* NPM, Package.json
* Debugging

Global Requirements
---------------

#### Grunt CLI 

` $ sudo npm install -g grunt-cli `

Commands
--------

#### Install Dependencies

` $ npm install `

#### Run Tests

` $ npm test `

#### Run App

` $ npm start `

#### Stop Server

` $ npm stop `

#### Run Development Environment

` $ npm run dev `

#### Run Debug Environment

` $ npm run debug `
